# LifeThreads TikTok / Reels Demo Scripts

## Script 1: Gallery vs Memory Wall

Hook:
```text
Your gallery stores photos. This app connects the memories behind them.
```

Shots:
- Start with normal photo gallery scroll.
- Cut to LifeThreads wall.
- Add one memory card.
- Connect it to another card with a rope.
- Open memory detail.

Voiceover:
```text
I wanted memories to feel alive, not buried in a camera roll. So I built LifeThreads: a private memory wall where photos, stories, places, and moments hang together. You can connect memories and write why they belong together.
```

CTA:
```text
Would you use this for travel, family, or personal memories?
```

## Script 2: Couple Timeline

Hook:
```text
Imagine your relationship memories as a living wall.
```

Shots:
- Add first date memory.
- Add trip memory.
- Add small text note.
- Connect memories with reason: "The first trip after everything changed."

Voiceover:
```text
Every relationship has moments that are connected. LifeThreads lets you build a private wall for them, with photos, notes, places, and the reasons each memory matters.
```

CTA:
```text
Send this to someone you would build a memory wall with.
```

## Script 3: Travel Wall

Hook:
```text
This is not a travel album. It is a map of what the trip felt like.
```

Shots:
- Create travel memory from photo.
- Show date/location prefill.
- Switch to map view.
- Open cinematic detail.

Voiceover:
```text
LifeThreads turns trips into connected chapters. Add photos, dates, locations, stories, and links between moments. Your memories stay local-first on your device.
```

CTA:
```text
Which trip would you add first?
```

## Script 4: Private by Design

Hook:
```text
Not every memory needs to be posted.
```

Shots:
- Show privacy card in Settings.
- Create private memory.
- Export backup.
- Show wall again.

Voiceover:
```text
LifeThreads is built for private memories. No social feed, no public profile, no account required in the current version. Your memory wall is yours.
```

CTA:
```text
Private memory apps should feel this personal.
```

## Script 5: The Emotional Product Demo

Hook:
```text
I am building an app for the memories you actually feel.
```

Shots:

- Open memory detail.

```text
A photo is not just a file. It is a place, a person, a feeling, and a moment in your life. LifeThreads is my attempt to make those memories feel alive again.
```
